## How to create initial files

Run command:

```shell
# LANG (ts|js, cs|csharp)
npm run create LANG YYYY-DD
```

The below files will be created:

For Typescript:

- {lang}/src/{YYYY}/files/input-day-{DD}.ts
- {lang}/src/{YYYY}/day-{DD}.ts

For C#:

- csharp/{YYYY}/day-{DD}.cs
- files/{YYYY}/{YYYY-DD}
